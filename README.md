# Personal Portfolio - Pritha S

This repository contains a personal portfolio website built with HTML, CSS, and JavaScript.

## Project Structure

- `index.html` - Main portfolio page with all sections like Hero, About, Education, Skills, Experience, Projects, Certifications, Contact, and Footer.
- `style.css` - Dark glassmorphism theme with modern gradients, animations, responsive styles, and hover effects.
- `script.js` - Typing text effect, scroll reveal animations, sticky navbar, smooth section highlighting, and mobile menu toggling.
- `document.empty` - Contains a single-file version of the complete portfolio (HTML+CSS+JS in one file).

## Features

- Clean, futuristic UI with glowing tech background
- Fully responsive (mobile, tablet, desktop)
- Glassmorphism + gradient design
- Animated text and section reveals
- Sticky navbar, smooth scrolling, active link highlighting
- Skills progress bars and project cards

## Setup & Run (Local)

### Option 1: open directly
1. Open `c:\Users\prith\OneDrive\Desktop\portfolio\index.html` in your browser.

### Option 2: VS Code + Live Server
1. Open the folder in VS Code: `File` -> `Open Folder...` -> `c:\Users\prith\OneDrive\Desktop\portfolio`
2. Install Live Server extension if not installed.
3. Right-click `index.html` -> `Open with Live Server`.

## How to edit

- For layout and styling changes: modify `style.css`.
- For animation behavior and nav interactions: modify `script.js`.
- For content change: edit `index.html`.

## Notes

- The site is built with only vanilla web technologies (no frameworks required).
- Keep all files together in the same folder so relative links remain valid.
